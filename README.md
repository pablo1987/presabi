# PRESABI S.A.S. - Web institucional (Astro + Tailwind)

Single page institucional para **PRESABI S.A.S.** enfocada en medicina domiciliaria y servicios de salud.

## Requisitos

- Node.js 20+
- npm 10+

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Build de producción

```bash
npm run build
npm run preview
```

## Imágenes obligatorias

Colocar estas imágenes en `public/images/` respetando exactamente los nombres:

- `4d40bf3b-26ae-4e5c-a07f-efe5b980f3c8.jpg` (logo)
- `beee7723-31ec-4743-9de3-929a2c67c8a9.jpg` (imagen principal)

> La estructura ya está preparada para agregar más imágenes en esa carpeta.

## Datos editables

En `src/pages/index.astro` podés editar fácilmente:

- WhatsApp
- Email
- Dirección
- Localidades de cobertura
- Textos institucionales
